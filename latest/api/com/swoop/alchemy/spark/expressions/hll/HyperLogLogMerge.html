case class HyperLogLogMerge(child: Expression, mutableAggBufferOffset: Int, inputAggBufferOffset: Int) extends TypedImperativeAggregate[Option[HyperLogLogPlus]] with NullableSketchAggregation with Product with Serializable

HyperLogLog++ (HLL++) is a state of the art cardinality estimation algorithm.

This version aggregates the "sketches" into a single merged "sketch" that represents the union of the constituents.
